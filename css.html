Materi Lengkap tentang CSS

Pengenalan CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web. CSS memungkinkan Anda untuk mengontrol tata letak, warna, font, dan elemen visual lainnya pada halaman web.

Sintaks Dasar

Sintaks CSS terdiri dari selektor dan deklarasi. Berikut adalah contoh sintaks dasar:


    selector {
        property: value;
    }

    /* Contoh */
    h1 {
        color: blue;
        font-size: 24px;
    }

Selektor

Selektor digunakan untuk memilih elemen HTML yang akan diberi gaya. Berikut adalah beberapa jenis selektor:

  • Selektor Elemen: Memilih semua elemen dengan nama tag tertentu.
  • Selektor Kelas: Memilih elemen dengan kelas tertentu, diawali dengan titik (.)
  • Selektor ID: Memilih elemen dengan ID tertentu, diawali dengan pagar (#)
  • Selektor Gabungan: Memilih elemen berdasarkan kombinasi selektor.

Properti CSS

Berikut adalah beberapa properti CSS yang umum digunakan:

  • color: Mengatur warna teks.
  • background-color: Mengatur warna latar belakang.
  • font-size: Mengatur ukuran font.
  • margin: Mengatur jarak luar elemen.
  • padding: Mengatur jarak dalam elemen.
  • border: Mengatur garis tepi elemen.
  • display: Mengatur cara elemen ditampilkan (block, inline, flex, dll).

Contoh Penggunaan CSS

Contoh 1: Mengubah Warna Teks


        <style>
            p {
                color: red;
            }
        </style>

        <p>Ini adalah paragraf dengan teks berwarna merah.</p>

Contoh 2: Mengatur Latar Belakang


        <style>
            body {
                background-color: lightblue;
            }
        </style>

Contoh 3: Mengatur Margin dan Padding


        <style>
            .box {
                margin: 20px;
                padding: 15px;
                background-color: white;
                border: 1px solid black;
            }
        </style>v
