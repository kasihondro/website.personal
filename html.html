Materi Lengkap tentang HTML

Pengenalan HTML

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat dan menyusun konten di web. HTML memberikan struktur dasar untuk halaman web dan memungkinkan penambahan teks, gambar, video, dan elemen multimedia lainnya.

Struktur Dasar HTML

Setiap dokumen HTML memiliki struktur dasar sebagai berikut:


    <!DOCTYPE html>
    <html lang="id">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Judul Halaman</title>
        </head>
        <body>
            <h1>Judul Utama</h1>
            <p>Ini adalah paragraf.</p>
        </body>
    </html>

Elemen Penting dalam HTML

Berikut adalah beberapa elemen penting dalam HTML:

  • <h1> hingga <h6>: Elemen judul, dari yang paling besar (<h1>) hingga yang paling kecil (<h6>).
  • <p>: Elemen paragraf.
  • <a>: Elemen tautan (link) yang digunakan untuk menghubungkan ke halaman lain.
  • <img>: Elemen gambar untuk menampilkan gambar.
  • <ul> dan <ol>: Elemen daftar tidak terurut (unordered list) dan terurut (ordered list).
  • <div>: Elemen kontainer untuk mengelompokkan elemen lain.
  • <span>: Elemen inline untuk mengelompokkan bagian dari teks.

Atribut dalam HTML

Atribut digunakan untuk memberikan informasi tambahan tentang elemen HTML. Atribut biasanya ditulis di dalam tag pembuka. Contoh atribut yang umum digunakan:

  • class: Menentukan kelas untuk elemen.
  • id: Menentukan ID unik untuk elemen.
  • href: Menentukan URL untuk tautan.
  • src: Menentukan sumber untuk gambar.
  • alt: Menyediakan teks alternatif untuk gambar.
