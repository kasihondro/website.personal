Materi Lengkap tentang JavaScript

Pengenalan JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif. JavaScript dapat digunakan untuk mengontrol elemen HTML, berinteraksi dengan pengguna, dan membuat aplikasi web yang dinamis.

Sintaks Dasar

Berikut adalah beberapa sintaks dasar dalam JavaScript:


    // Komentar satu baris
    /* Komentar 
       beberapa baris */

    var nama = "John"; // Deklarasi variabel
    let umur = 30; // Deklarasi variabel dengan let
    const pi = 3.14; // Deklarasi variabel konstan

    console.log(nama); // Menampilkan output di konsol

Tipe Data

JavaScript memiliki beberapa tipe data dasar:

  • String: Teks, contoh: "Hello World"
  • Number: Angka, contoh: 42
  • Boolean: Nilai benar/salah, contoh: true atau false
  • Array: Kumpulan nilai, contoh: [1, 2, 3]
  • Object: Kumpulan pasangan kunci-nilai, contoh: { nama: "John", umur: 30 }

Fungsi

Fungsi adalah blok kode yang dapat dipanggil untuk menjalankan tugas tertentu. Berikut adalah contoh fungsi dalam JavaScript:


    function sapa(nama) {
        return "Halo, " + nama + "!";
    }

    console.log(sapa("John")); // Output: Halo, John!

Contoh Penggunaan JavaScript

Contoh 1: Menampilkan Pesan


        alert("Selamat datang di halaman ini!");

Contoh 2: Menghitung Jumlah Dua Angka


        function tambah(a, b) {
            return a + b;
        }

        console.log(tambah(5, 10)); // Output: 15

Contoh 3: Menggunakan Array


        var angka = [1, 2, 3, 4, 5];
        angka.forEach(function(item) {
            console.log(item);
        });

Kesimpulan

JavaScript adalah bahasa pemrograman yang sangat kuat dan fleksibel. Dengan memahami dasar-dasar JavaScript, Anda dapat mulai membuat aplikasi web yang interaktif dan dinamis.
