Materi Lengkap tentang PHP

Pengenalan PHP

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk mengembangkan aplikasi web. PHP sering digunakan untuk membuat situs web dinamis dan dapat berinteraksi dengan basis data.

Sintaks Dasar

Berikut adalah beberapa sintaks dasar dalam PHP:


    <?php
        // Komentar satu baris
        /* Komentar 
           beberapa baris */

        $nama = "John"; // Deklarasi variabel
        $umur = 30; // Deklarasi variabel

        echo $nama; // Menampilkan output
    ?>

Tipe Data

PHP memiliki beberapa tipe data dasar:

  • String: Teks, contoh: "Hello World"
  • Integer: Angka bulat, contoh: 42
  • Float: Angka desimal, contoh: 3.14
  • Boolean: Nilai benar/salah, contoh: true atau false
  • Array: Kumpulan nilai, contoh: array(1, 2, 3)
  • Object: Instance dari kelas, contoh: $obj = new ClassName();
  • NULL: Tidak ada nilai, contoh: $var = NULL;

Kontrol Alur

PHP mendukung kontrol alur seperti pengkondisian dan perulangan. Berikut adalah contohnya:


    // Pengkondisian
    if ($umur >= 18) {
        echo "Anda sudah dewasa.";
    } else {
        echo "Anda masih remaja.";
    }

    // Perulangan
    for ($i = 0; $i < 5; $i++) {
        echo $i . " ";
    }

Fungsi

Fungsi adalah blok kode yang dapat dipanggil untuk menjalankan tugas tertentu. Berikut adalah contoh fungsi dalam PHP:


    function sapa($nama) {
        return "Halo, " . $nama . "!";
    }

    echo sapa("John"); // Output: Halo, John!
